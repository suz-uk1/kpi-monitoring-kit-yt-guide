## 3. 初回実行の「承認」について（不安を潰す）
初回だけ、Googleから権限承認の画面が出ます。

- これは **スプレッドシートを編集したり、Discord/Emailに送信する**ために必要な手続きです。
- 承認はいつでも取り消せます（Googleアカウントのセキュリティ設定から）。

承認画面が出たら：
1) 「詳細」→「（安全でないページ）に移動」→許可
2) 許可後、もう一度 `Setup` / `Update & Send` を実行
