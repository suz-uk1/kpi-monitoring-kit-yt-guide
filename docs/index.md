# YouTube週次KPIモニタ（手入力Pro） セットアップガイド

このページは購入者向けの導入ガイドです。
テキストで手順をすべて説明しつつ、「画面はこう見えるはず」の目印としてスクショも掲載します（順次追加）。

---

## 0. 全体の流れ（最短5分）
1. テンプレをコピー（自分のGoogle Driveに複製）
2. Apps Script を貼り付け（コードを入れる）
3. `KPI Kit → Setup (create tabs)` を実行（初回だけ）
4. `Input_Weekly` に週1行入力 → `KPI Kit → Update & Send`

---

## 1. テンプレをコピー

### 1-1) テンプレを開く
購入したZIPの `01_TEMPLATE` にあるURLから、テンプレを開きます。

（スクショ：テンプレURLを開いた直後の画面）

※このテンプレには見本として **ダミーの入力が1行**入っています。
運用を始めるときは、`Input_Weekly` の2行目（ダミー）を削除してから使ってください。

### 1-2) 自分のDriveにコピーする
スプレッドシート上部メニューから：
- ［ファイル］→［コピーを作成］

ここで作られたコピーが「あなたの運用版」です。

（スクショ：ファイル→コピーを作成）

---

## 重要：テンプレの見本表示と、運用開始の手順について

このテンプレ（販売用シート）には、購入者が「完成イメージを掴む」ために
- シートが作られている
- `Input_Weekly` にダミー行が1行入っている

という **見本** が入っています。

ただし、テンプレ自体に入っているスクリプトは「見本を作るためのもの」であり、
購入後に運用するファイルでは **ZIPに同梱の `main_clean.gs`（本体）を貼り付けて実行**してください。

以降の手順は、運用するコピー（あなたのDriveに作成したコピー）に対して行います。

## 2. Apps Script を貼り付け（コードを入れる）

### 2-1) Apps Script を開く
コピーしたスプレッドシートで：
- ［拡張機能］→［Apps Script］

（スクショ：拡張機能→Apps Script）

### 2-2) main_clean.gs を全置換して保存
ZIP内の `02_APPS_SCRIPT/main_clean.gs`（または `.gs.txt`）を開き、
中身をすべてコピーして、Apps Script側の `main_clean.gs` を全置換 → 保存します。

（スクショ：Apps Scriptのエディタで貼り付け）

---

## 3. 初回実行の「権限承認」について

初回だけ、Googleから権限承認の画面が出ます。

### 3-1) なぜ承認が必要？
このキットは Apps Script で以下を行います：
- スプレッドシート内に必要なシートを作成・更新（Home/Weekly_Reportなど）
- （任意で設定した場合のみ）Discord/Emailへの送信

そのため、Googleが「このスクリプトに許可を与えますか？」という確認を出します。

### 3-2) 「安全ではないページ」と表示される理由
Apps Scriptを自分で貼り付けた直後は、Google側が
「これはGoogleが審査・配布した公式アドオンではなく、ユーザーが自分で作成/貼り付けたスクリプトですよ」
という意味で警告を出します。

ここで表示される文言は不安になりやすいですが、
- **自分のGoogle Drive上のスプレッドシートに紐づいたスクリプト**であること
- スクリプトの中身（`main_clean.gs`）がZIP内で確認できること

を前提に、内容を確認して進めるための仕組みです。

### 3-3) 承認の進め方（手順）
承認画面が出たら：
1) 「詳細」→「（安全ではないページ）に移動」
2) 権限の内容を確認して「許可」
3) 許可後、スプレッドシートに戻って `Setup` / `Update & Send` をもう一度実行

（スクショ：承認画面の例）

### 3-4) 不安な場合の進め方（安全に段階を踏む）
- まずは `Config` の Discord/Email は **空のまま** でOKです
  - 空の場合、送信は行いません
- 先に `Setup` → `Update & Send` を動かして、Home/Weekly_Reportが更新されることだけ確認できます

---

## 4. 週次の使い方（毎週）

1) `Input_Weekly` に週1行入力
2) `KPI Kit → Update & Send`
3) `Home` が更新、`Weekly_Report` に週報テキストが出ます

（スクショ：Input_Weekly / Update & Send / Weekly_Report）

---

## 5. 送信設定（任意）

- Discord：`Config` の `discord_webhook_url`
- Email：`Config` の `email_to`

設定後、`Update & Send` を実行してください。
